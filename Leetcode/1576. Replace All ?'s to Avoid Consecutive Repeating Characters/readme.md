# Algorithm Idea

The approach iterates through the string and replaces each '?' with one of the characters 'a', 'b', or 'c' such that no consecutive characters are the same.

The solution ensures this by checking the character before and after the current '?' and picking a character that does not match either.


# Complexity

- Time: O(N)

- Space:O(1)
