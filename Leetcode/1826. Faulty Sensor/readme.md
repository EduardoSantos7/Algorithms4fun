# Algorithm Idea

The algorithm identifies the first mismatch between two sensor readings and checks if shifting one sensor's data by one position aligns it with the other, to determine which sensor is faulty—or returns -1 if the faulty one can't be determined.

# Complexity

- Time: O(N)

- Space:O(1)
