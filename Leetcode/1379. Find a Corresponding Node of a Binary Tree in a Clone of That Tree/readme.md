# Algorithm Idea

Use DFS/BFS to traverse the tree and compare the node you are visiting with the target, it they are the same value then return the reference from the cloned tree.

# Complexity

DFS)

- Time: O(n)

- Space:O(n)

BFS)

- Time: O(n)

- Space:O(n)
