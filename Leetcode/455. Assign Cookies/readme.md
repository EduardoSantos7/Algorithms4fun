# Algorithm Idea

Sort both arrays and give the biggest cookie to child who wants the biggest and pop up the cookie and the child. Instead of pop you can also use pointers, or apply this greedy algorithm backwards.

# Complexity

- Time: O(nlog)

- Space:O(logn)
