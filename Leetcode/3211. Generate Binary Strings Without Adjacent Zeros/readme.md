# Algorithm Idea

Approach 1)

Generate the binary string by couting from 0 to N, converting the number to binary and the right size, then check if there are 2 consecutive zeroes.

Approach 2)

Backtracking

# Complexity

Approach 1)

- Time: O(2**N * N)

- Space:O(2**N * N)

Approach 2)

- Time: O(2**N)

- Space:O(N)
