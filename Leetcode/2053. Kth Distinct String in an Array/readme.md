# Algorithm Idea

Count the frequencies of each word.
Iterate all the word and for each that freq is 1 decrease k by 1 and if k is 0 return the word.
If after that iteration you didn't return, then return an empty string

# Complexity

- Time: O(n)

- Space:O(n)
