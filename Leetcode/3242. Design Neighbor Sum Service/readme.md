# Algorithm Idea

In the constructor of the service:

Calculates the sums of adjacent and diagonal neighbors for each cell in the grid.
Stores these sums in two lists (self.adjacent and self.diagonals) for quick access

# Complexity

- Time: O(N^2)

- Space:O(N)
