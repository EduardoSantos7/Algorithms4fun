# Algorithm Idea

Construct an array where each index will hold the answer for the length of the string requested. The value of the index will be the sum of the end - zeros and end - ones where end is the current index.

# Complexity

- Time: O(heigh) // O(n)

- Space:O(heigh) // O(n)
